# NurGruz - Сайт грузоперевозок и грузчиков в Бишкеке

## О проекте
Одностраничный сайт для компании грузоперевозок. Адаптивный, современный дизайн, форма обратной связи с отправкой в Telegram.

## Технологии
- React + Vite
- CSS (свои стили)
- Node.js + Express (бэкенд для формы)
- Telegram Bot API

## Функции
- Главный экран с призывом к действию
- Блок "О нас" с фото
- Услуги (6 карточек)
- Автопарк (8 машин с характеристиками)
- Отзывы (слайдер)
- Контакты + форма заявки
- Адаптив под все устройства
